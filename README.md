# Drum Kit 🥁

An interactive Drum Kit web application built using HTML, CSS, and JavaScript.  
Play different drum sounds using mouse clicks or keyboard keys.

---

## Features

- Play drum sounds on mouse click
- Play drum sounds on keyboard key press
- Interactive and responsive UI
- Realistic drum audio effects
- Event handling using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation & Event Listeners)

---

## What I Learned

Through this project, I learned:

- How to use `addEventListener()` in JavaScript
- DOM selection using `querySelectorAll()`
- Handling mouse and keyboard events
- Using `switch` statements for different actions
- Playing audio files dynamically with `new Audio()`

---

## How to Run the Project

1. Download or clone the repository
2. Open `index.html` in your browser
3. Click the buttons or press keys `w a s d j k l`
